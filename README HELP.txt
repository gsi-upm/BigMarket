Recommended IDE: Eclipse IDE for Java EE Developers,  http://www.eclipse.org/downloads/packages/eclipse-ide-java-ee-developers/keplersr1
Recommended Java Servlet and JavaServer Pages technologies implementation: Apache Tomcat, http://tomcat.apache.org/. Follow this tutorial to install and configure it properly, http://tomcat.apache.org/tomcat-6.0-doc/setup.html
Recommended source forge: Github, Egit is a eclipse pligin to import Git projects directly http://www.eclipse.org/egit/download/ (Install it in Eclipse from: Help>Install new Software)
If you want to run the project like a Web project you must create a new server runtime in Eclipse: http://www.eclipse.org/webtools/jst/components/ws/M4/tutorials/InstallTomcat.html

Now you have all the necessary tools to run, study and change BigMarket

-Import the git project in Eclipse
-Right click on index.html (WebContent>index.html)>Run as> Run on server
-Select Choose an existing server (created in the previously)
